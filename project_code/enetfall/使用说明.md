具体使用方法和csibench数据集基本一样

但由于这个数据集的一些特点，还是会有一些差别



1.这个数据集不用--use_ema,用了之后效果反而变差，原因可能是样本量不够多，导致每一次的best相互之间差别都很大，综合起来的权重反而不好用。



2.这个数据集样本量也比较少，所以没有设置同分布的测试集



3.运行命令大致是

```
python run_baselines_compare_enetfall.py --model (要测的模型) --save_dir (路径)
```
直接运行python run_baselines_compare_enetfall.py就是测所有写在参数表中的模型，并将结果保存在写在参数表的路径上



保存的内容也基本上和之前的差不多




另外还有一个csibench的文件夹里面装的是优化后的csibench的dataset_loader.py,经过测试发现会有一些些提升，你也可以替换尝试一下,看看效果怎么样
